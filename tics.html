SUB-2 / Diseño Digital

Plantel 22 - Grupo 606

Propósito de la Materia

Aprender y aplicar los fundamentos de la maquetación web y el diseño digital. Utilizando lenguajes de marcado como HTML, comprendemos la estructura, elementos y funcionamiento de una página web, integrando tablas, recursos multimedia y aplicando principios de jerarquía visual y psicología del color.

💻 Problema Laboral – Sitio Web Escolar

Publicado: 4 mar

Este portafolio es la evidencia de este proyecto. Consiste en programar un sitio web completo interconectando 6 páginas HTML. El reto es aplicar correctamente la paleta de colores, tipografía adecuada, lograr un equilibrio visual y entregar el código alojado en un repositorio de GitHub antes del examen parcial.
